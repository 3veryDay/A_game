## 1. 요구사항 분석
### 핵심 기능
- 2분 빠른 페이스 / 2분 느린 페이스 구간 음악 자동 조절
- 러닝 타이머와 음악 템포 동기화
- 사용자 맞춤형 플레이리스트 관리

### 세부 요구사항
- BPM(Beat Per Minute) 기반 음악 선택 알고리즘 (추후 추가)
- 사용자 운동 패턴에 맞는 음악 추천 기능 (추후 추가)
- **러닝 구간별 음악 자동 전환**
- 페이스별 음악 추천, 선택 알고리즘

### 사용자 시나리오 도출
1. 초기 사용자 시나리오
- 앱 최초 실행
- 개인 음악 라이브러리 연동
- 러닝 모드 설정 ( 인터벌 종류, 운동 시간, 휴식 시간, 반복 횟수 )

인터벌 훈련을 할 때 정해야 할 주요 요소들은 다음과 같아!  

### 1. **인터벌 종류**  
   - **고강도 인터벌 트레이닝(HIIT)**: 짧고 강한 운동(예: 전력 질주) + 짧은 휴식  
   - **템포 인터벌**: 중강도로 지속적인 페이스 유지(예: 마라톤 페이스)  
   - **언덕 인터벌**: 언덕에서 강도 높은 달리기 + 평지에서 회복  
   - **시간 기반 인터벌**: 일정 시간(예: 30초 전력 질주 + 1분 걷기)  
   - **거리 기반 인터벌**: 특정 거리(예: 400m 전력 질주 + 200m 조깅)  

### 2. **운동 시간(강도 구간)**  
   - 얼마나 강하게, 얼마나 길게 운동할지 결정  
   - 예: 30초, 1분, 3분 등  

### 3. **휴식 시간(회복 구간)**  
   - 운동 후 쉬는 시간 (완전 휴식 or 가벼운 운동)  
   - 강도가 높을수록 휴식을 길게 설정 (예: 1:1 or 1:2 비율)  

### 4. **반복 횟수(세트 수)**  
   - 초보자는 4~6세트, 숙련자는 8~12세트  
   - 목표에 따라 세트 수 조절 (지구력 vs. 스피드)  

### 5. **강도(페이스, 심박수 등)**  
   - 최대 심박수의 몇 %로 뛸지 설정 (예: 80~90%)  
   - 속도를 일정하게 유지할 것인지 체크  

### 6. **목표(목적 설정)**  
   - 지구력 향상, 속도 증가, 체지방 감소 등 목표 설정  
   - 예: 마라톤 대비 훈련, 축구 체력 훈련 등  

어떤 목표로 인터벌 훈련을 하려고 해? 💪🔥
